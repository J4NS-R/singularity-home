Singularity.net.za

Welcome to George Rautenbach's corner of the web.

Singularity is George's digital service provider arm. Here's some of its utility:

  • Mankind.wtf's frontend and backend
  • George's website's backend
  • Link shortening service
  • Personal use video conferencing
  • Personal custom cloud host
  • Media server
  • P2P node
  • Fallback email server
  • Sundry PHP apps webserver (see the projects list for links)
  • Automatic backups runner
  • Continuous integration and deployment server
  • SSL certificate management
  • DNS virtual domain waypoint
  • Many, many more facets...

Singularity was named after the currently-hypothesised point in time at which the intelligence of machines will surpass that of mankind (wiki). Being one of the greatest philosophical issues in computer science, George deemed it a very apt name. Learn more about George.
